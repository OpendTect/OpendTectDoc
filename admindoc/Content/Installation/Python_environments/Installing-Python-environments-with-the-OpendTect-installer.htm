Installing Python environments with the OpendTect installer

When running the OpendTect installer there is the option to install the Python environments. The installer will auto select the Miniconda3 environment and also auto select the Math Kernel Library (MKL) and CUDA 10 when selecting the Machine Learning plugin.


Figure: Install Python environments with the OpendTect installer

If you have your own miniconda3 or anaconda3 environment you might want to consider to install the OpendTect Python environments from YAML files instead.
