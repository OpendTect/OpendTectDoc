Presentation Maker Plugin

Introduction

python-pptx is a Python library for creating and updating PowerPoint (.pptx) files. The OpendTect 'Presentation Maker' plugin uses this library to create a PowerPoint presentation from scene, window or desktop screenshots. Up to OpendTect 6.4 it was required to install Python and all required Python libraries separately to get the Presentation Maker Plugin working. From OpendTect 6.6 and onwards OpendTect will come with Python and libraries. Therefore it is no longer needed to install Python separately. Of course this is still possible and might be what you want.

python-pptx installation

General information can be found in the installing chapter of the python-pptx documentation.

Windows

On Windows, three packages need to be installed. First of all python itself, then lxml and finally python-pptx. The three packages have to be installed in the given order and installation instructions are given below. At the time of writing python-pptx requires Python 2.7, 3.3, 3.4, or 3.6.

Python

  1. Before you start, create a new folder to store the Python installation, e.g. C:\apps.
  2. Download the Python installer from the Python download page. At the time of writing the version 3.7.4 is the preferred version to install. Choose Windows x86-64 executable installer for Windows 64 bits and Windows x86 executable installer for Windows 32 bits.
  3. During installation, choose C:\apps as the installation folder and select the option to add Python to the PATH environment variable. If you don't get this option, or missed it, edit the PATH environment variable and add the following:
    C:\apps\Python37\;C:\apps\Python37\Scripts;
    More information on setting environment variables on Windows can be found at this Computer Hope page.

lxml

The package python-pptx requires the lxml tools. At the time of writing the most recent version is 4.4.2.

  1. Download lxml library installer from this the lxml download page. Choose lxml-4.4.2-cp37-cp37m-win32.whl for Windows 32 bits and lxml-4.4.2-cp37-cp37m-win_amd64.whl for Windows 64 bits.
  2. Open the Command Prompt (press Windows+R, type in cmd, and hit enter), change the current directory into the download directory and run command:
    pip install lxml-4.4.2-cp37-cp37m-win_amd64.whl or
    pip install lxml-4.4.2-cp37-cp37m-win32.whl

python-pptx

  1. Open the Command Prompt (press Windows+R, type in cmd, and hit enter), and run command:
    pip install python-pptx

Linux

Start by checking whether Python has been installed.

  1. Open the terminal and enter: python --version. If it's installed, you'll see the version number. At the time of writing python-pptx requires Python 2.7, 3.3, 3.4, or 3.6.
  2. If Python is not installed, install using your distribution's package manager.
  3. First install lxml:
    pip install lxml
  4. Then install python-pptx:
    pip install python-pptx
