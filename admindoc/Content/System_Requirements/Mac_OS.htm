Mac OS

Minimum

  • Version: Mac OS X 10.10 and newer
  • CPU: Mac/Intel processor with 64-bit support
  • GPU: similar to the Intel/AMD platform graphics card
  • Memory: 8 GB of RAM
  • Storage: Hard Disk

Recommended

  • Version: Mac OS X 10.10 and newer
  • CPU: Mac/Intel processor with 64-bit support
  • GPU: similar to the Intel/AMD platform graphics card
  • Memory: 32 GB of RAM
  • Storage: SSD is best. Other options are Hard Disk and Network Drive.

For Machine Learning

  • This is still to be determined. This section will be updated when we release OpendTect 6.6 for Mac OS X.

Please note:

  • Machine Learning on GPU is not supported due to the absence of Nvidia cards for Mac OS systems.
  • Mac/PowerPC support is NOT available.
  • A 3-button mouse is highly recommended.
  • OpendTect v6.4 supports Mac OS X from 10.10 onwards.
