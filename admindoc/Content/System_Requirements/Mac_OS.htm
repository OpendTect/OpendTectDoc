Mac OS

Minimum

  • Version: For OpendTect 6.4: Mac OS X 10.10 and newer

  • CPU: Mac/Intel processor with 64-bit support

  • GPU: Basic Intel or AMD graphics card, f.i. the Intel HD Graphics 4000.

  • Memory: 8 GB of RAM

  • Storage: Hard Disk

Recommended

  • Version: For OpendTect 6.4: Mac OS X 10.10 and newer

  • CPU: Mac/Intel processor with 64-bit support

  • GPU: Intel or AMD graphics card

  • Memory: 32 GB of RAM

  • Storage: SSD is best. Other good options are Hard Disk and Network Drive.

For Machine Learning

  • This is still to be determined.

Please note:

  • Machine Learning on GPU is not supported due to the absence of Nvidia cards for recent Mac OS X systems.

  • Mac/PowerPC support is NOT available.

  • A 3-button mouse is highly recommended.
