Configuring BatchHosts on Linux

On Linux the multi-machine communication is done using remote shell commands like rsh or ssh. In the BatchHoats file the default is already set to rsh. The user can also change this to ssh if needed. The list of computers to be used has to be entered in the following format.

 

Host.10.IP Address: 192.168.0.87

Host.10.Hostname: dgb87

Host.10.Display Name: Superbox (8x Xeon E5-2665 24x2x 5.40 GHz, 64 bits, OpenSuse 13.2)

Host.10.Platform: lux64

Host.10.Data Root: /auto/d46/ODData
