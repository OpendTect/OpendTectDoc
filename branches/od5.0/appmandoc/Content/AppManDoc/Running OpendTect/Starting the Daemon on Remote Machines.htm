Starting the Daemon on Remote Machines

Start the daemon on every remote machine by selecting

 

C:\ProgramData\Microsoft\Windows\Start Menu\Programs\OpendTect\5.0.0\Remote Processing Service

 

This tool will add a notification icon to the system tray and pop up this window:

 

 

Press 'Start'... the remote machines are now ready to be used as multi-machine processing nodes. The service can also be controlled by right-clicking on the system tray icon and selecting appropriate menu item like start, stop or exit:
