Import Wells

Purpose: Importing well/checkshot data to a new or existing OpendTect project.

 

Theory: Importing well data into OpendTect from SEG-Y, general ASCII or LAS files.

 

Software: OpendTect

 

Workflow: All import functions are accessed from the 'Survey' menu in the main toolbar (Survey > Import)

 

  1. Well data can be imported to OpendTect in various formats; ASCII, VSP and Simple Multi-Well. There is also functionality to support Bulk import (Bulk Tracks, Bulk Logs, Bulk Markers and Bulk Time-Depth Models).

     

  2. For importing well data from ASCII file, first specify the type of data (tracks, logs or markers).

     

  3. For well tracks, specify the input file, header and format (column order in data file). If the well is vertical or no well track file is available, uncheck the well track check-box and provide the surface coordinate (in meters), KB elevation and total depth (TD).

     

  4. In the second section, specify the depth to time model, header and format definition. Here, you also have the option to specify whether its a check-shot data or normal well data. If no depth to time model is available, uncheck the check-box and specify a temporary, constant model velocity (by default 2000 m/s).

     

  5. Finally, in the advanced/optional settings the surface coordinate can be set (if different from the starting coordinate in the well track). A number of additional fields are available for further well information such as the Well ID, Operator and State & County. All information filled out here will also show up in the well information window in the Well Manager.

     

  6. For importing various well logs, go to Survey > Import > Wells > ASCII > Logs... or via the Manage Wells window and press Import. Select the input LAS log file; specify (change) the Depth interval to load and toggle either 'TVDSS' or 'MD' values. Finally, select the logs that you wish to import, and specify to which well you wish to add these logs.

     

  7. Similarly, for importing the well markers, in the Manage Wells window press Edit markers . In the newly opened Edit Well Markers window press Import and select the input ASCII file, define the file header and format (column order in data file) and specify if you want to keep any existing markers or replace them completely.

     

  8. If well data is available in VSP (SEG-Y) format, first select and Examine the input SEG-Y file in the SEG-Y tool window. Then, specify if the data is in time or depth. In the second section, select to which well the VSP data should be added and specify a name for the new log. The output interval can be limited if desired.

     

  9. The Simple Multi-Well import option allows simultaneous creation of several vertical wells, either manually or by reading data from a file (click Read file.. in the bottom left corner of the Simple Multi-Well Creation window, specify the file and format in the popup dialog).
