Install demo License

Plugins to OpendTect can either run using a license server or using demo (evaluation) licenses. This second case is called "node-locked license installation". Use the following window to specify the path of license files that were given to you if you do not wish to use a license server:

 

 

There should be one line in this window per license vendor. Select the file(s) and press OK. You may be prompted to restart. For all questions related to the plugins please refer to your license vendor.
