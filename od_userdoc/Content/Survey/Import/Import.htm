4.3 Import

The Survey > Import drop-down menu is used to import data to OpendTect.

Direct data exchange with Schlumberger's Petrel is available via PetrelDirect plugin (part of OpendTect Pro)PetrelDirect plugin (part of OpendTect Pro)

Direct data exchange with Landmark's SeisWorks and Schlumberger's GeoFrame-IESX is available via Workstation Access plugin by ARK CLSWorkstation Access plugin by ARK CLS.
