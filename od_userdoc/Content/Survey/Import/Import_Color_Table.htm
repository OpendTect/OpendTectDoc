4.3.2 Import Color Table

An OpendTect color table can be imported via Survey > Import > Color Table... or Manage Color Tables window > Import button.

There are two ways to import a color table:

  • Other user option is used when it's possible to browse to the other user's home directory. Navigate to the folder and type in DTECT_USER name (if any).
  • Choose the File option if other users' home directories are not accessible. The color tables created by OpendTect users are stored in a settings_coltabs.DTECT_USER file (DTECT_USER = OpendTect username) that is located in the user's home directory $HOME/.od/.

The default OpendTect color tables are stored in a ColTabs file that is located in the OpendTect installation directory e.g. /home/your-username/OpendTect/.0/data/ or C:\Program Files\OpendTect\.0\data\ .
