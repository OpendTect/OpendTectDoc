4.3.13.2 Import Zero-Offset VSP

A zero-offset VSP data can be imported for a selected well via Survey > Import > Wells > VSP (SEG-Y)... First browse and locate the input file. Then in the Import SEG-Y DataImport SEG-Y Data window check the quick scan results and press Next when done.

In the pop-up Import Zero-offset VSP window, select the type of input Z values (TWT, TVDSS or MD), select the well to which the VSP log should be added and press OK to import the log.
