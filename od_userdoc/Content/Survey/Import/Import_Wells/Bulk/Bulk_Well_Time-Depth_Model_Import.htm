4.3.13.4.4 Bulk Well Time-Depth Model Import

Time-depth models can be imported for several wells in bulk from a single ASCII file via Survey > Import > Wells > Bulk > Depth/Time model... The specification for the input data is similar to the single well importsingle well import.

Click Define... to pop up the Format Definition dialog:

You will have the option to select either the well Name or UWI (Unique Well Identifier). And also to set depth as either MD or TVDSS. You may also toggle on the 'Stop reading at' choice and set a value here.
