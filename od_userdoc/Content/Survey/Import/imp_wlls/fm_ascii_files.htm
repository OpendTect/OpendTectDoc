From Ascii Files

 

 

Well depths in OpendTect are always referenced using their Measured Depth. The alignment with the seismic data in done using the track (deviation survey) data and time-depth and/or checkshot data. The track data provides the relation between lateral coordinates, TVDSS depths and Measured Depths. The checkshot data and time-depth data provides the relation between Measured depths and travel times (TWT). Other formats can be used during import, but will be converted.
