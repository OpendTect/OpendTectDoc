Export Ascii 2D Horizons

 

 

2D Horizon Export Window

 

 

For exporting a 2D horizon, you begin by selecting the horizon and the required 2D lines. If a header is needed it can be specified as either single line or multi line. Further, if the undefined parts in the data have to be written a standard value needs to be specified. Similarly, the Z value can be explicitly output in milliseconds irrespective of the original format used in the survey, it will be stored in seconds otherwise.
