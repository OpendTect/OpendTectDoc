4.4.7 Export Probability Density Functions

Probability Density Functions (PDFs) can be exported as ASCII files in Icon Science's RokDoc format via Survey > Export > Probability Density Functions > ASCII (RokDoc)...

In the Export Probability Density Function window: select a PDF; type an output file name (to save to Survey Data Root folder) or provide a full path by clicking Select; and press Export.

A successful export is confirmed with a message, and the export window stays open to export more PDFs.
