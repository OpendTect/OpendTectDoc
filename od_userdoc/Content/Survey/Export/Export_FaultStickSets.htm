4.4.2 Export FaultStickSets

Fault stick sets can be exported as ASCII files via Survey > Export > FaultStickSets > ASCII...

In the Export FaultStickSet window: select a fault stick set; specify the output format; type an output file name (to save to Survey Data Root folder) or provide a full path by clicking Select; select a coordinate system of the output file (the option is available only if the current survey has a defined projection based coordinate system), and press Export.

A successful export is confirmed with a message and an option to either export more fault stick sets or close the export tool.
