Export PickSets & Polygons

Pick sets and polygons can be exported as ASCII files via Survey > Export > PickSets/Polygons > ASCII...

In the Export PickSets/Polygons window: select a pick set or a polygon; type an output file name (to save to survey root folder) or provide a full path by clicking Select; and press Export. The output file contains 3 columns: X/Y coordinates and Z (time or depth) values.

A successful export is confirmed with a message and an option to either export more pick sets/polygons or close the export tool.
