Export Mute Functions

Mute functions can be exported as ASCII files via Survey > Export > Mute Functions > ASCII...

In the Export Mute Function window: select a mute definition; choose coordinates format; type an output file name (to save to survey root folder) or provide a full path by clicking Select; and press Export. The output file contains 4 columns: X/Y or Inl/Crl coordinates, offset and Z (time or depth) values.

A successful export is confirmed with a message, and the export window stays open to export more mute functions.
