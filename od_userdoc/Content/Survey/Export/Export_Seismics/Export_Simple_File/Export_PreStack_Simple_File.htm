4.4.8.2.3 Export PreStack Simple File
In OpendTect, you can also export a simple seismic Ascii or Binary file from Pre-Stack 3D seismic data via Survey > Export > Seismic > Simple File > Prestack 3D.
  • Input Data Store: Select the Pre-Stack seismic to export.
  • Volume Subselection: A part of the line can also be sub-selected. If the entire line is meant to be exported: after clicking on Select..., define the number and time ranges. Leave the values default or select All in the trace subselection window.
  • Scale values: The data can be scaled in the output. The output will be calculated with: Output = Factor * Input + Shift
  • Null traces: Null traces can be discarded or left inside the line (Pass).
  • Output a position for every trace: If Yes then the output will contain a short header information. If No, no header will be added.
  • (If Yes selected in the previous option) Position in file will be: In the output file, the position information can either be XY locations or Inline/crossline numbers
  • Include: Optionally you can select to include Offset and/or Azimuth information.
  • Put sampling info in file start: Select Yes to allow the sampling information to appear at the beginning of the file.
  • File type: Select the appropriate output file type: ASCII or Binary.
  • Output file: Select/write the output file location.
