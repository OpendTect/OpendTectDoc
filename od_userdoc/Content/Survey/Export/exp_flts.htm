Export Faults

Faults can be exported as ASCII files via Survey > Export > Fault > ASCII...

In the Export Fault window: select a fault from the current OpendTect survey; specify the output format; type an output file name (to save to survey root folder) or provide a full path by clicking Select; and press Export.

If the specified output file exists A successful export is confirmed with a message and an option to either export more faults or close the export tool.
