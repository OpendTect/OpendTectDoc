4.1.2.4 Read from Survey Setup file

This option for initial setup will prompt you to provide a Survey Setup file like the ones created in the root of the OpendTect projects as .survey files (see: https://doc.opendtect.org/7.0.0/doc/od_userdoc/Default.htm#survey/export/export_survey_setup.htm)

Examine the file to check its contents and press OK

All of the information contained in the setup file will be used to create the survey ranges, X/Y transformations etc.
