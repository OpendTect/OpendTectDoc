4.1.2.2 Scan Navigation Data

This method asks for an Input Geometry File for either Single or Multiple Lines:

The file must contain X, Y, trace number and shot point number. For Multiple Lines the file and the format definition must also contain the Line Name (usually as column 1).

Examine the file and set the number of header lines, if present, and Define the format, then click OK:

Once completed, click Continue.

The Edit Survey Parameters window will now appear and, after checking the setup and entering a valid Z-range, pressing OK will load the survey into the dataroot.
