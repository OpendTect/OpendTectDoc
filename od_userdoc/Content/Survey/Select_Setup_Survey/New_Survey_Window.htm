4.1.2 New Survey Window

To launch the survey setup window select New in survey selection window.

Depending on whether you have OpendTect or OpendTect Pro installed, one of the following windows will appear on your screen:

Almost all setup options are available for both the Free and Pro versions, with the exception of: Scan VDS file and Scan ZGY file.

Survey name: In the text area specify the OpendTect survey name.

Available data: Toggle on the data type(s) to be included in the survey - any combination of: 3D, 2D and Wells. Note: Select only 3D, if the survey contains only a 3D type data set. Select both 2D and 3D, if the survey is to contain both 2D and 3D type data sets. If the survey contains only 2D type data set, select only 2D. Selection type here affects the tree structure and what functions and attributes are available to you once the survey is operational.

Initial setup: Determines how you set up the survey ranges and coordinates. Each Initial Setup option will be dealt with in its own sub-topic, links below.

Z Domain: Can be in time or depth (for depth, define here the unit as m or ft)
