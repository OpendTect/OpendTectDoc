4.5.12.1.1.1 Copy - Input Selection

The input selection for copy cube can be done from teh list of stored cubes or from:

SEG-Y directly

PetrelDirect link

The stored cubes in this list can be, as is standard on all input selection UIs:

renamed

locked/unlocked, or

deleted.

Users also have the option to filter the selection using the topmost dialogue box.
