4.5.6 Manage FaultStickSets

This option allows the management of faultsticksets. The faultstickset management window can be launched either by pressing the icon from manage toolbar, or from menu bar i.e. Survey > Manage> FaultStickSet...

This window will contain the list of faultstickset interpreted and saved in the OpendTect project as shown below.

Select the desired fault, then use the various tools:

is used to change the current disk location of selected faultstickset.

is used to rename the selected faultstickset.

locks the selected faultstickset from interpretation.

is used to trash (delete) the selected faultstickset.

is used to set a faultstickset as default.

copies the faultsticksets.

The top filter is used to filter-out the objects with selected names. For instance, to display all faultsticksets that start with letter F use "F*".
