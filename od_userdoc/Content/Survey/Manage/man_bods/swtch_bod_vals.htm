Switch Body Values

The icon activates a window with dual functionality.

 

Firstly, if you have constructed a regional body using two horizons and a fault, creating a kind of 'compartment', this tool allows you to 'flip' the body to the 'negative' of the one constructed.

 

Secondly, previous versions of OpendTect had bodies with many different formats. This tools can also be used to 'convert' these various formats into the standard format for 4.6.
