1.2 Release Notes

This is the user documentation for release OpendTect v - an open source post-processing, and seismic interpretation system created by dGB Earth SciencesdGB.

OpendTect is released via the internet. Users can download the software from the download page of the dGB websiteOpendTect website. It will run without license protection.

OpendTect v is released under a GNU GPL license.

 

Under the GNU GPLGNU GPL license OpendTect is completely free-of-charge, including for commercial use.

OpendTect Pro, the commercially extended version of OpendTect is protected by FlexNet license management software. OpendTect Pro offers extra functionality and allows commercial users to extend the system with additional (closed source) commercial plugins that can either be purchased or leased. These commercial plugins for OpendTect Pro are also protected by the same FlexNet license managing software. To obtain a license key for OpendTect Pro and the plugins please contact dGB at info@dgbes.com.

Under the Academic licenseAcademic license agreement universities can get free licenses for OpendTect Pro and commercial plugins for R&D and educational purposes.

OpendTect is currently supported on the following platforms:

  • Linux (64bit)
  • Windows 10 and 11 (64bit)
  • Mac OS X 10.14 (Mojave) and 10.15 (Catalina)

For more information please see the System Requirements.
