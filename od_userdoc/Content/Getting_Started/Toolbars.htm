2.2 Toolbars

Many OpendTect processes can be launched most easily by clicking the corresponding icon. We distinguish two types of icons:

  • Icons that perform one specific action, e.g. moves the current display element (inline, crossline, Z-slice) N-steps forward.
  • Icons that open a menu with multiple options, e.g. launches either the 2D attributes -, or the 3D attributes - definition window.

Icons are organized in toolbars. Some toolbars are always present, others only appear when their action is appropriate. For example the display element toolbar (upper left corner) only appears when a display element in the tree is selected and its content varies with the type of display element.

Please note that if you are missing icons in your version of OpendTect then you are probably running a version without (some of) the commercial plug-ins.
