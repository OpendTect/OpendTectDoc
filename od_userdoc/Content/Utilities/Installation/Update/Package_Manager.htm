9.3.1.1 Package Manager

The last window of the wizard is the OpendTect Package Manager (see above figure). Multiple items can be selected from the list by checking the boxes (or not). Optionally, the relevant package combination could also be selected from the top list box.

The installation manager will automatically recognize the previously installed version at the selected path and will prompt it in the Installed version field.
