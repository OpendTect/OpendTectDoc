9.4.8 Show HostID

This option scans your system, and lists relevant Host information (HostID, Name, Operating System, User Name).

This information can be copied using the icon.

Additionally, on Windows, accessing the HostID of the machine can be done via the LM Tools (available via the Start Menu or directly from ..\OpendTect\6.6.0\bin\win64\lm.dgb\lmtools.exe):
