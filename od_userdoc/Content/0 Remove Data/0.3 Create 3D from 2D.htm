Create 3D from 2D

The Create 3D from 2D option creates 3D volumes from 2D line-sets.

• Input Data Set/Attribute: Select 2D data set and data (attribute) for conversion into 3D.

• Type of Interpolation: Nearest trace, as the name implies uses the nearest trace for the interpolation. The FFT based interpolation is a 3D Fourier transform of Inlines and cross-lines set in the Interpolation window. Optionally, check the Re-use interpolated traces; this option will use interpolated values for the next iterations. Finally, set the Maximum velocity to pass (m/s).

• Output Cube: Name for new 3D volume.

• Volume Subselection: Optionally, set the sub-selection using inlines/cross-lines and z-ranges.
