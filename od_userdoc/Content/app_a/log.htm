Log

Name

Log -- an attribute that returns a well log value.

Description

This attribute takes the value form an input well log and returns this value throughout the volume at the corresponding Z value.

Input Parameters

Firstly, the 'Input Well' is chosen and, secondly, one of the associated logs from the auto-filled 'Select Log' list.
