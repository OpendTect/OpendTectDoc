3.7.12 Set Z Values

A 'Z' value surface attribute (see Store Z as attributeStore Z as attribute) can be used to shift a horizon or completely change its 'Z' positions using the Set Z values option.

Specifying values as Relative (deltas) will shift the horizon; infact the software adds the attributes 'Z' values to the 'Z' values of horizon to achieve this shift. Absolute is used while completely changing the 'Z' values of the horizon to the 'Z' values of the surface attribute. Specification of units of 'Z' values (i.e. in 'milliseconds' or 'seconds') is also required.
