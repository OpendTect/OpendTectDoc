5.5.1.3.2 Seismic Attributes

The seismic attribute selection/definition is comparable to the main attribute windowmain attribute window: the same attributes are available. An attribute can be selected in the list of Available types and add to the Defined attributes using the button. The parameters of the attribute have to be specified. Synthetic seismic generated from the models can be used as input data. All the listed attributes are not necessary using synthetic seismic.
