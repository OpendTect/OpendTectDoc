6.1.8.1.1 Simple Time-Depth Model

In the Time-Depth Conversion windows, it is possible to chose 'Simple Time-Depth Model' as one of the conversion options.

Select will allow choice of a stored Simple Time-Depth Model from your existing database:

The models are by default stored in ...\<survey name>\Models\

Create/Edit provides a choice between two options in a drop-down:

Both options use the same UI:

Create is blank, ready for your input. Edit opens the currently selected model. Entries may be renamed, locked, deleted or set as favourites, as in all other selection windows.
