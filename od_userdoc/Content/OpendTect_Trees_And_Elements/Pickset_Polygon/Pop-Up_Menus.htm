3.6.5 Pop-Up Menus

Pickset/Polygon Element Pop-up Menu

This is the menu right-click menu accessed from the main Pickset/Polygon element when at least one pickset/polygon is already loaded (see pictures below).

It contains the following menu items.

  • Add …: load a stored pickset/polygon in the 3D scene and tree.
  • New: Create a new pickset/polygon (see 3.6 Pickset & Polygon3.6 Pickset & Polygon).
  • Display only at sections: Display picks only intersecting the displayed elements (plane(s) and/or horizon(s)) in the 3D scene. This mode enables picking in a new location without being distracted by previously picked picks throughout the survey box.
  • Display in Full: Display all the picks of the pickset or the full polygon within the survey box.
  • Merge Sets: (for picksets only) Merge stored picksets into a new pickset. A window pops up and the sets can be selected. A name for the output set should be defined. Picksets can also be merged from the Pickset ManagerPickset Manager.
  • Save changes: The pickset/polygon changes can be saved and reloaded at any time during the building process.

When more than one pickset/polygon are loaded in the tree, the menu has additional entries:

  • Show all items: Display all the picksets/polygons from the tree in the 3D scene.
  • Hide all items: Unselect all the picksets/polygons from the tree. They are no more displayed in the 3D scene.
  • Remove all items: Remove all the picksets/polygons from the tree

Pickset / Polygon sub-elements Pop-up Menu

If at least one pickset or polygon is loaded in the tree, then the following options are available from the right-click menu (see picture below):

  • Calculate Volume: In OpendTect, an estimated volume can be computed from a polygon to a given surface. The velocity default is set to 3000 m/s. Negative thicknesses can either be discarded or taken into account.
  • Convert to body: (for pickset only) Convert the pickset into a bodybody.
  • Create Body: (for polygon only) Create a body using the polygon as a constraining area. It requires a top and bottom horizon between which the body would be created. This feature is only enabled if SSIS pluginSSIS plugin is loaded (or licensed). F
  • Close polygon: (only when interpreting a new polygon) During, and at the end of a picking session, picksets should be stored.
  • Display:
    • Display only at sections: Display picks only intersecting the displayed elements (plane(s) and/or horizon(s)) in the 3D scene. This mode allows to pick new locations without being distracted by previously picked picks throughout the survey volume.
    • Properties: In this window the Type, Size and Color of the pick markers in the 3D scene can be set. The type Arrow is also automatically used when the pick is given directional information in the Set directions option under the pickset pop-up menu.
    • Set direction: (for pickset only) Display direction, guided by the SteeringCube/attribute. This helps to understand the geological dips and fluid flow. It is assigned by setting a direction to each pick based on dip and azimuth information (attributes). In the pop-up window (see below), specify either a SteeringCube or two attributes providing the polar dip and azimuth in degrees. A velocity of 2000 m/s will be used in time survey to convert the dip from degrees to µs/m if the dip angle data is read from a stored cube instead of the dip angle attribute. Do not forget after setting the directions to save you pickset and change the display type to "Arrow".

An example of setting direction (black arrows) to a pickset.

When setting the direction for a given pickset, you can select to get the direction from a SteeringCube or from the Azimuth and Dip attributes (stored or on the fly) (see picture below)

  • Save/Save As: Either overwrite the stored input by using option Save or store as new pickset / polygon by using Save As option.
  • Lock / Unlock: Lock the selected object. It prevents accidental removing, moving or displaying data on the object. Clicking "Unlock" enables editing again.
  • Remove: Remove pickset/polygon from tree.

Removing any unsaved pickset/polygon will result in complete loss of the unsaved interpretation.

  • Export to Google KML: Export selected polygon to a Google KML file. When selected, the following export window is launched. Fill in the output KML parameters and write/select the output file location. Press the 'Ok' button to export the polygon in the selected location. The feature will prompt an additional conversion dialog if the conversion settings for the survey are not defined. For further information, please refer to the Survey SelectionSurvey Selection section.
