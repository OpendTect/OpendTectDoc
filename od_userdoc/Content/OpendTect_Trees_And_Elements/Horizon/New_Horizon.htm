3.7.7 New Horizon

3D Horizon > New right-click menu in the Tree allows user to create new 3D horizons using one of the following:

  • Auto and Manual Tracking: interpret a horizon using 3D Volume Auto-tracking, Section Auto-tracking and/or Manual Draw
  • With Constant Z: create a horizon with constant Z value
  • Track Unconformity: (requires commercial Dip-Steering plug-in)
