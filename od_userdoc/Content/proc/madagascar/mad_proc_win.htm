Madagascar Processing Window

The Madagascar processing window can be launched from the OpendTect toolbar by pressing the Madagascar icon.

 

 

Select the input cube to be processed, and then choose a program or combination of programs. Programs are organized into groups of programs. Once one program is selected, a description of program's functions are shown in the neighboring frame.

 

 

The different steps, as well as a synopsis, of the computation are provided. The descriptions of each program are available on the Madagascar site.
