Body Shape Painter

The body shape painter is used to define the inside and outside values for an OpendTect body. The options for 'Inside Fill Type' are shown in the image below. The same options are available for 'Outside Fill Type':

 

 

 

Fill Types:

  • Constant Value: Single, user-defined value:

 

  • Previous step: takes the values of the previous step in the Volume Builder setup (ie: a stored volume):

 

  • Undefined Value: Equivalent to 'transparency'

 

 

One example of use would be: if one wants to create a salt velocity cube, the values inside can be filled with a salt velocity and outside can be set to previous step. If no other step exists the undefined value is written.
