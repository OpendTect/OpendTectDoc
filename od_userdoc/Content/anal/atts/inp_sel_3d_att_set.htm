5.1.5.1 Input Selection for 3D Attribute Sets

Select from the stored data or from the list of defined attributes in the "active" attribute set.

In case the input data is multi-component, it is possible to choose from the available components as shown below (or include ALL).

The Filter section allows to quickly find the right input. e.g type *S will look for all attributes/cubes started with S like Similarity.
