4.5.10 Manage PickSets & Polygons

In this window, all available picksets are shown. These picksets can be moved to a different location, they can be renamed, locked / unlocked, removed or set as default using the icons on the right.

Several pickset can be merge together using Merge Pickset Button.

The top filter is used to filter-out the objects with selected names. For instance, to display all picksets that start with letter S use "S*".
