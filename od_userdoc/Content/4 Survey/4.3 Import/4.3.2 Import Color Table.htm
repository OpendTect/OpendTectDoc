4.3.2 Import Color Table

An OpendTect color table can be imported via Survey > Import > Color Table....

In the Import Color Tables window select to Import from:
  • Other user: to import a color table from another OpendTect user
  • File: to import a color table from file

The default OpendTect color tables are stored in a ColTabs file that is located in the OpendTect installation directory /root/OpendTect/6.0.0/data/.

The color tables created by OpendTect users are stored in a settings_coltabs.user (user = OpendTect username) file that is located in the user home directory $HOME/.od/.
