Introduction

 

This document describes various work flows in the OpendTect base package plus plugins. We describe the purpose, what software is needed (OpendTect base only or base plus commercial plugins), and how to do it. The workflows are described in simple steps, outlining what needs to be done.

 

Where relevant, links to the OpendTect User Documentation, dGB Plugins User Documentation, the OpendTect and dGB websites are given in order to provide further information. For these latter links to work, you will need access to the internet.
