4.3 Building OpendTect

Building in CMAKE

Run CMAKE and select the folders for the source code and where to build the binaries (for example C:\dev\od for both). Press configure, it will give an error, do not worry, and click OK. Fill in the QTDIR with the path to your installed QT folder. Give the whole path including the folder msvc2017_64 and configure again.

Press OK on the error warning, and now give the path to your OSG 3.6.3 release directory and configure again.

If it says configuring done, press the generate button to build the solution files. Now you can open the OpendTect.sln file!
