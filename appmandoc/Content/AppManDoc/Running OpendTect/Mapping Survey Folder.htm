Mapping Survey Folder

It is assumed that the survey folder is in a server which has to be shared and mapped on all host machines to be used for multi-machine processing.

 

If the survey folder is on the C drive on the server e.g. C:\Surveys then map the C drive from this machine to other host machines by choosing an available drive letter ( like C:, D: S: etc). So the survey folder now becomes S:\Surveys or F:\Surveys on each of the machines depending on drive letter that is used.
