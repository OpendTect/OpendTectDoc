General

Users need to be aware of only a few things to be able to use OpendTect. Things become slightly more complicated when multiple users carry out shared projects.

 

The specific user settings and current survey are stored in the '.od' hidden directory in the user's home directory.

 

The file names are .settings and .survey, or settings.$DTECT_USER and survey.$DTECT_USER if the DTECT_USER environment variable is set (see Project Sharing).
