The GMT Link

To activate the GMT link, GMT needs to be installed. The download site is http://gmt.soest.hawaii.edu.

 

After installation, make sure the GMT binary directory ('bin') is in the user's PATH. For many purposes, the user will want a PostScript viewer to be installed, too. Make sure you have a good association for postscript files in place.

 

More general information can be found in the OpendTect User Documentation: http://www.opendtect.org/500/doc/od_userdoc/Default.htm#proc/gmt/gmt.htm%3FTocPath%3DProcessing%7CGMT%7C_____0
